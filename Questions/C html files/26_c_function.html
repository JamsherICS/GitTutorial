A function is a block of code that performs a specific task. Functions in C provide modularity and reusability to programs. Here's how you use functions in C:

  1. Function Declaration:
    • A function declaration specifies the function's name, return type, and parameters (if any).
  2. Function Definition:
    • A function definition contains the actual implementation of the function.
  3. Function Call:
    • To execute a function, you need to call it from another part of the program.

Here's an example of defining and using a function in C:

#include <stdio.h>

// Function declaration
int add(int a, int b);

int main() {
    int result;

    // Function call
    result = add(5, 3);
    printf("Result: %d\n", result);

    return 0;
}

// Function definition
int add(int a, int b) {
    return a + b;
}

In this example:

  • We declare a function named add that takes two integer parameters and returns an integer.
  • We define the add function to return the sum of its parameters.
  • We call the add function from the main function and print the result.
