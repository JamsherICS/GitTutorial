The printf() function in C is used to print formatted output to the standard output device, typically the console. Its syntax is as follows:


// printf() function syntax
printf("format_string", arg1, arg2, ...);

format_string: A string that specifies the format of the output. It may contain format specifiers like %d, %f, %s, etc., which are replaced by the corresponding arguments.

arg1, arg2, ...: Arguments to be formatted and inserted into the format string. They must match the format specifiers in the format string.


// Example usage of printf()
int num = 10;
printf("The value of num is: %d\n", num);

This will print: The value of num is: 10
