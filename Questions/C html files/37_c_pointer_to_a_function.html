A pointer to a function in C holds the address of a function instead of holding a regular data value. This allows us to pass functions as arguments to other functions, return functions from other functions, or store functions in data structures like arrays.

Here's how you declare and use a pointer to a function:

#include <stdio.h>

// Function to add two numbers
int add(int a, int b) {
    return a + b;
}

// Declaration of pointer to a function
int (*ptr)(int, int);

int main() {
    // Assigning address of 'add' function to 'ptr'
    ptr = &add;

    // Calling the function using pointer to function
    int result = (ptr)(3, 4);
    printf("Result: %d\n", result);

    return 0;
}

In this example, we declared a function add to add two numbers. Then, we declared a pointer to a function named ptr. We assigned the address of the add function to ptr, and then called the function using the pointer.
