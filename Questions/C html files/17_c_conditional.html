1. if statement:

It is used to execute a block of code if a specified condition is true.

2. else if statement:

It is used to specify a new condition to test, if the first condition is false.

3. else statement:

It is used to execute a block of code if the condition of the if statement and all else if statements are false.

#include <stdio.h>

int main() {
    int num = -5;

    // Check if 'num' is greater than 0
    if (num > 0) {
        printf("%d is positive.\n", num);
    } 
    // If 'num' is not greater than 0, check if it's less than 0
    else if (num < 0) {
        printf("%d is negative.\n", num);
    } 
    // If 'num' is neither greater than 0 nor less than 0, it must be equal to 0
    else {
        printf("%d is zero.\n", num);
    }
    
    return 0;
}

In this code:
We declare and initialize a variable num with the value -5. We use if, else if, and else statements to check different conditions. If num is greater than 0, we print that it is positive. If num is less than 0, we print that it is negative. If neither of the above conditions is true, we print that num is zero.
