1. Local Variables:

  • Scope: Local variables are declared inside a block or function, and their scope is limited to that block or function. They are accessible only within the block or function in which they are defined.
  • Lifetime: Local variables are created when the block or function is entered and destroyed when the block or function exits. They exist only for the duration of the block or function in which they are declared.
  • Accessibility: Local variables are not visible outside the block or function in which they are defined. They cannot be accessed or modified by other functions or blocks in the program.

Example:

  #include <stdio.h>
  
  void function() {
      int x = 10; // Local variable
      printf("Local variable x: %d\n", x);
  }
  
  int main() {
      function();
      // printf("%d\n", x); // Error: x is not accessible here
      return 0;
  }

2. Global Variables:

  • Scope: Global variables are declared outside of all functions, typically at the beginning of the file or at the top of the program. They have global scope, meaning they are accessible from any part of the program, including all functions in all files of the program.
  • Lifetime: Global variables are created when the program starts and exist for the entire duration of the program's execution. They are destroyed when the program terminates.
  • Accessibility: Global variables can be accessed and modified from any part of the program. They are visible to all functions and blocks within the program, allowing for easy sharing of data between different parts of the program.

Example:

    #include <stdio.h>
    
    int globalVar = 20; // Global variable
    
    void function() {
        printf("Global variable: %d\n", globalVar);
    }
    
    int main() {
        printf("Global variable: %d\n", globalVar);
        function();
        globalVar = 30; // Modify global variable
        printf("Modified global variable: %d\n", globalVar);
        return 0;
    }

In summary, the main differences between local and global variables in C lie in their scope, lifetime, and accessibility. Local variables are limited to a specific block or function, have a shorter lifetime, and are not visible outside their scope. Global variables, on the other hand, are accessible throughout the program, have a longer lifetime, and can be accessed and modified from any part of the program.
