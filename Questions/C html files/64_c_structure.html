In C, a structure is a user-defined data type that allows you to group together variables of different data types under a single name. Each variable within a structure is called a member or field. Structures provide a way to organize related data items more logically and efficiently.

Example:

  #include <stdio.h>

  // Define a structure named "Person"
  struct Person {
  char name[50];
  int age;
  float height;
 };

  int main() {
  // Declare a variable of type "Person"
  struct Person person1;

  // Access and modify structure members
  strcpy(person1.name, "John");
  person1.age = 30;
  person1.height = 5.9;

  // Print structure members
  printf("Name: %s\n", person1.name);
  printf("Age: %d\n", person1.age);
  printf("Height: %.2f\n", person1.height);

  return 0;
 }

output:

  Name: John
  Age: 30
  Height: 5.90

Now, let's discuss how to pass a structure to a function:

Passing by Value:

  • When you pass a structure to a function by value, a copy of the entire structure is made and passed to the function.
  • Any modifications made to the structure inside the function do not affect the original structure in the calling function.

Passing by Reference:

  • When you pass a structure to a function by reference (using a pointer), you pass the address of the structure to the function.
  • This allows the function to directly modify the original structure in the calling function.

Example:

#include <stdio.h>

// Define a structure named "Person"
struct Person {
    char name[50];
    int age;
    float height;
};

// Function prototypes
void displayByValue(struct Person p);
void displayByReference(struct Person *p);

int main() {
    // Declare a variable of type "Person"
    struct Person person1 = {"John", 30, 5.9};

    // Pass structure by value
    displayByValue(person1);

    // Pass structure by reference
    displayByReference(&person1);

    return 0;
}

// Function to display structure by value
void displayByValue(struct Person p) {
    printf("Displaying by value:\n");
    printf("Name: %s\n", p.name);
    printf("Age: %d\n", p.age);
    printf("Height: %.2f\n", p.height);
}

// Function to display structure by reference
void displayByReference(struct Person *p) {
    printf("Displaying by reference:\n");
    printf("Name: %s\n", p->name);
    printf("Age: %d\n", p->age);
    printf("Height: %.2f\n", p->height);
}

Output:

  Displaying by value:
    Name: John
    Age: 30
    Height: 5.90

  Displaying by reference:
    Name: John
    Age: 30
    Height: 5.90
