Multi-Dimensional Arrays in C

A multi-dimensional array is an array that contains other arrays as its elements. It is essentially an array of arrays, where each element of the main array is itself an array. Multi-dimensional arrays are commonly used to represent tables, matrices, or other structured data with multiple dimensions.

Syntax:

data_type array_name[size1][size2]...[sizeN];
          
          data_type: Specifies the type of data that the array will hold (e.g., int, float, char).
          array_name: Specifies the name of the array.
          size1, size2, ..., sizeN: Specifies the sizes of each dimension of the array.

Example:

Here's an example of a 2D array (matrix) in C:

#include <stdio.h>
          
          int main() {
              int matrix[3][3] = {
                  {1, 2, 3},
                  {4, 5, 6},
                  {7, 8, 9}
              };
          
              /* In this example:
                 - matrix is a 2D array with 3 rows and 3 columns.
                 - Each element of the matrix is accessed using two indices: matrix[row_index][column_index]. */
          
              return 0;
          }

Accessing Elements

To access elements of a multi-dimensional array, you need to specify the indices corresponding to each dimension.

int value = matrix[row_index][column_index];
          row_index: Specifies the index of the row.
          column_index: Specifies the index of the column.

Initializing Multi-dimensional Arrays

Multi-dimensional arrays can be initialized using nested curly braces to specify the values for each dimension.

int matrix[2][3] = {
              {1, 2, 3},
              {4, 5, 6}
          };

Passing Multi-dimensional Arrays to Functions

When passing multi-dimensional arrays to functions, you need to specify the sizes of all dimensions except the first one.

void printMatrix(int rows, int cols, int matrix[rows][cols]) {
              // Function implementation
          }

Dynamic Allocation of Multi-dimensional Arrays

Multi-dimensional arrays can be dynamically allocated using pointers and malloc() or calloc() functions.

int **matrix;
          matrix = malloc(rows * sizeof(int *));
          for (int i = 0; i < rows; i++) {
              matrix[i] = malloc(cols * sizeof(int));
          }

Usage

Multi-dimensional arrays are commonly used to represent data structures like matrices, grids, tables, and images. They provide an efficient way to organize and access data in multiple dimensions, allowing for complex data manipulation and algorithms in C programming.
