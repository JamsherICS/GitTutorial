Bitwise Shift Operators in C

Bitwise shift operators are used in C (and many other programming languages) to shift the bits of a binary number left or right. There are two main bitwise shift operators:

Left Shift Operator (<<):

This operator shifts the bits of a number to the left by a specified number of positions. It effectively multiplies the number by 2 for each shift to the left. The syntax is:

result = value << n;

Here, value is the number whose bits are to be shifted, and n is the number of positions to shift the bits.

Right Shift Operator (>>):

This operator shifts the bits of a number to the right by a specified number of positions. It effectively divides the number by 2 for each shift to the right. The syntax is:

result = value >> n;

Here, value is the number whose bits are to be shifted, and n is the number of positions to shift the bits.

Usage example:

#include <stdio.h>
  
  int main() {
      int num = 10; // binary representation: 0000 1010
  
      // Left shift
      int shifted_left = num << 2; // Shifts bits of num two positions to the left
      // Result: 0010 1000 (equals 40 in decimal)
  
      // Right shift
      int shifted_right = num >> 1; // Shifts bits of num one position to the right
      // Result: 0000 0101 (equals 5 in decimal)
  
      printf("Left shifted result: %d\n", shifted_left);
      printf("Right shifted result: %d\n", shifted_right);
  
      return 0;
  }

Bitwise shift operators are commonly used in tasks like implementing multiplication and division by powers of 2, manipulating bit-level representations of data, and optimizing code for certain operations.
