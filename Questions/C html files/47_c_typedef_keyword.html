Imagine you have a friend named John. But instead of always calling him "John," you decide to give him a nickname, like "Johnny." Now, whenever you want to refer to your friend, you can just use the nickname "Johnny" instead of his full name "John." It's the same person, just with a different name you've chosen.

In C, typedef does something similar. It lets you give a nickname (or alias) to a data type. For example, instead of always writing int for integer variables, you could use typedef to create a nickname like Int32. So, whenever you need an integer, you can just use Int32 instead of int.

typedef int Int32; // Int32 is now an alias for int

Int32 number = 42; // Same as writing: int number = 42;

Using typedef, you can also create nicknames for more complex data types, like structures or arrays, to make your code easier to understand and maintain.
