In C programming, there are several file modes that you can use when opening files:

  • "r" (Read Mode):
    • Opens a file for reading.
    • If the file doesn't exist or cannot be opened, fopen() returns NULL.
    • The file pointer is positioned at the beginning of the file.

  • "w" (Write Mode):
    • Opens a file for writing.
    • If the file exists, its contents are overwritten. If the file doesn't exist, a new file is created.
    • fopen() will return NULL if the file cannot be opened.
    • The file pointer is positioned at the beginning of the file.

  • "a" (Append Mode):
    • Opens a file for appending.
    • If the file doesn't exist, a new file is created.
    • The file pointer is positioned at the end of the file.
    • Data written to the file is always appended to the end.

  • "r+"" (Read/Write Mode):
    • Opens a file for both reading and writing.
    • The file must exist, or fopen() will return NULL.
    • The file pointer is positioned at the beginning of the file.

  • "w+"" (Read/Write Mode):
    • Opens a file for both reading and writing.
    • If the file exists, its contents are overwritten. If the file doesn't exist, a new file is created.
    • The file pointer is positioned at the beginning of the file.

  • "a+"" (Read/Append Mode):
    • Opens a file for both reading and appending.
    • If the file doesn't exist, a new file is created.
    • The file pointer is positioned at the end of the file.
    • Data written to the file is always appended to the end.

Below is an example of how to use these file modes in C:

#include <stdio.h>

int main() {
    FILE *file;

    // Example: Reading from a file
    file = fopen("example.txt", "r");
    if (file == NULL) {
        printf("Error opening file for reading!\n");
        return 1;
    }
    printf("File opened successfully for reading.\n");
    // Read operations go here
    fclose(file);

    // Example: Writing to a file
    file = fopen("example.txt", "w");
    if (file == NULL) {
        printf("Error opening file for writing!\n");
        return 1;
    }
    printf("File opened successfully for writing.\n");
    // Write operations go here
    fprintf(file, "This is an example line written to the file.\n");
    fclose(file);

    // Example: Appending to a file
    file = fopen("example.txt", "a");
    if (file == NULL) {
        printf("Error opening file for appending!\n");
        return 1;
    }
    printf("File opened successfully for appending.\n");
    // Append operations go here
    fprintf(file, "This line is appended to the file.\n");
    fclose(file);

    // Example: Reading and writing to a file
    file = fopen("example.txt", "r+");
    if (file == NULL) {
        printf("Error opening file for reading and writing!\n");
        return 1;
    }
    printf("File opened successfully for reading and writing.\n");
    // Read and write operations go here
    fclose(file);

    return 0;
}
