In C, you can declare and initialize arrays as follows:

type arrayName[size] = {value1, value2, ..., valueN};

Where:

  • type specifies the data type of the elements in the array.
  • arrayName is the name of the array.
  • size is the number of elements the array can hold.
  • value1, value2, ..., valueN are the initial values of the array elements.

Here's an example of declaring and initializing an array of integers in C:

#include <stdio.h>

int main() {
    int numbers[5] = {10, 20, 30, 40, 50}; // Declare and initialize an array of integers
    
    return 0;
}

In this example, we declare and initialize an array of integers named numbers with a size of 5. The elements of the array are initialized with the values 10, 20, 30, 40, and 50, respectively.
