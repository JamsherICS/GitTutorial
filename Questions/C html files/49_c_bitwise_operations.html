Bitwise operations in C are operations that manipulate individual bits of binary representations of integers. These operations work directly on the binary representations of integers, allowing you to perform tasks such as setting, clearing, or toggling specific bits, as well as performing logical operations at the bit level.

Common Bitwise Operators:

  • Bitwise AND (&):
    • Performs a logical AND operation on corresponding bits of two operands.
    • The result is 1 if both bits are 1; otherwise, it's 0.
  • Bitwise OR (|):
    • Performs a logical OR operation on corresponding bits of two operands.
    • The result is 1 if at least one of the bits is 1; otherwise, it's 0.
  • Bitwise XOR (^):
    • Performs a logical XOR (exclusive OR) operation on corresponding bits of two operands.
    • The result is 1 if the bits are different; otherwise, it's 0.
  • Bitwise NOT (~):
    • Performs a logical NOT operation on each bit of the operand.
    • It flips each bit, changing 0 to 1 and 1 to 0.
  • Left Shift (<<):
    • Shifts the bits of the left operand to the left by a specified number of positions.
    • Each shift moves the bits to the left and fills in with zeros on the right.
  • Right Shift (>>):
    • Shifts the bits of the left operand to the right by a specified number of positions.
    • For unsigned types, it fills in with zeros on the left.
    • For signed types, it fills in with the sign bit (0 for positive, 1 for negative) on the left, which preserves the sign of the number.

Example:

#include <stdio.h>

int main() {
    unsigned int x = 5; // Binary representation: 0000 0101
    unsigned int y = 3; // Binary representation: 0000 0011

    // Bitwise AND
    printf("x & y = %u\n", x & y); // Result: 0000 0001 (1 in decimal)

    // Bitwise OR
    printf("x | y = %u\n", x | y); // Result: 0000 0111 (7 in decimal)

    // Bitwise XOR
    printf("x ^ y = %u\n", x ^ y); // Result: 0000 0110 (6 in decimal)

    // Bitwise NOT
    printf("~x = %u\n", ~x); // Result: 1111 1010 (4294967290 in decimal, assuming 32-bit int)

    // Left Shift
    printf("x << 1 = %u\n", x << 1); // Result: 0000 1010 (10 in decimal)

    // Right Shift
    printf("x >> 1 = %u\n", x >> 1); // Result: 0000 0010 (2 in decimal)

    return 0;
}

Bitwise operations are often used in low-level programming, such as manipulating hardware registers, implementing data compression algorithms, or optimizing code for performance. They provide a powerful tool for working with individual bits and performing efficient bit-level computations.
