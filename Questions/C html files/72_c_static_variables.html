In C, a static variable is a type of variable that retains its value between function calls and exists for the entire duration of the program. Unlike automatic variables, which are created and destroyed every time a function is called, static variables are initialized only once, and their value persists across function calls within the same scope.

Example: 

 #include <stdio.h>
    
 void increment() {
     static int count = 0;  // Static variable
     count++;
     printf("Count: %d\n", count);
 }
 
 int main() {
     increment();  // Output: Count: 1
     increment();  // Output: Count: 2
     increment();  // Output: Count: 3
     return 0;
 }

Output:

  Count: 1
  Count: 2
  Count: 3
