File handling in C involves working with files to perform various input/output operations. It allows you to read data from files, write data to files, create new files, and modify existing files.

The key components of file handling in C include:

  • File Pointers: In C, file operations are performed using file pointers, which are variables that hold the memory address of a file stream. The FILE data type is used to declare file pointers. File pointers are essential for performing operations such as opening files, reading from files, writing to files, and closing files.
  • Opening Files: Before performing any operation on a file, it needs to be opened. The fopen() function is used to open files in C. It takes two arguments: the file path and the mode in which the file is to be opened (e.g., read, write, append). If the file does not exist, fopen() can create a new file.
  • Reading from Files: Once a file is opened for reading, data can be read from the file using functions such as fscanf(), fgets(), or fread(). These functions read data from the file and store it in variables or buffers for further processing.
  • Writing to Files: When a file is opened for writing, data can be written to the file using functions such as fprintf(), fputs(), or fwrite(). These functions allow writing data to the file from variables or buffers.
  • Closing Files: After performing operations on a file, it should be closed using the fclose() function. Closing a file releases the resources associated with it and ensures that all data is properly written to the file.
#include <stdio.h>

// Example of file handling in C
int main() {
    FILE *filePtr;
    filePtr = fopen("example.txt", "w");  // Open file in write mode
    fprintf(filePtr, "Hello, World!\n");  // Write data to file
    fclose(filePtr);  // Close file
    return 0;
}

In this example, we use the FILE pointer and functions like fopen, fprintf, and fclose to create a new file named "example.txt", write "Hello, World!" to it, and then close the file.
