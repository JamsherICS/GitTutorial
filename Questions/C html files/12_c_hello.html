In C, you can print "Hello, World!" to the console using the printf() function, which is declared in the <stdio.h> header file. Here's how you do it:

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

The printf() function formats and prints output to the standard output device, which is usually the console. In this case, it prints "Hello, World!" followed by a newline character \n.
