The main() function serves as the entry point of a C program. It is the starting point of program execution. When a C program is executed, the operating system calls the main() function first. The program's execution begins from the first statement inside the main() function and continues sequentially.

#include <stdio.h>

int main() {
    // Program starts execution from here
    printf("Hello, World!\n"); // Print "Hello, World!"
    return 0; // Return 0 to indicate successful program execution
}

int main(): This is the declaration of the main() function, which returns an integer value (int). It does not accept any arguments.

In C, main() should return an integer value, typically 0 to indicate successful execution.
