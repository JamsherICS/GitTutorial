Input can be read from the user in C using the scanf() function for formatted input or the gets() or fgets() functions for reading strings.

The scanf() function is used to read input from the standard input device (typically the keyboard) based on a specified format string. It takes the address of variables where the input will be stored as arguments.

Using scanf() function for simple input like numbers.

#include <stdio.h>

int main() {
    int num;
    printf("Enter a number: ");
    scanf("%d", &num);  // Reads an integer input from the user
    printf("You entered: %d\n", num);
    return 0;
}

Using fgets() function for reading strings, including spaces.

#include <stdio.h>

int main() {
    char name[50];
    printf("Enter your name: ");
    fgets(name, sizeof(name), stdin);  // Reads a string input from the user
    printf("Hello, %s\n", name);
    return 0;
}

For example, scanf("%d", &num); reads an integer input from the user and stores it in the variable num. The gets() function reads a line of text from the standard input and stores it as a string, but it is not recommended to use due to security vulnerabilities. The fgets() function is safer and is typically preferred for reading strings.
