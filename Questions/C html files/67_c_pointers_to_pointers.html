Pointers to pointers, also known as double pointers, are pointers that store the address of another pointer. In C, pointers to pointers provide a way to indirectly access and modify a pointer variable, enabling dynamic memory allocation and manipulation of data structures like arrays and linked lists. Here's an explanation of pointers to pointers:

Definition:

  • A pointer to a pointer is declared by using two asterisks (**) before the variable name.
  • Syntax: type **pointer_to_pointer;

Memory Representation:

  • Just like a regular pointer, a pointer to a pointer also holds a memory address.
  • However, instead of pointing to a single data object, it points to another pointer variable that, in turn, points to a data object.
  • This creates an additional level of indirection, allowing for more flexibility in memory management.

Usage:

  • Pointers to pointers are commonly used in scenarios where you need to dynamically allocate memory or modify pointer variables indirectly.
  • They are frequently used in functions that need to modify a pointer variable passed as an argument.
  • They are also used in dynamic data structures like linked lists, trees, and multi-dimensional arrays.

Example:

  #include <stdio.h>
  
  int main() {
      int num = 10;
      int *ptr = &num; // Pointer to an integer variable
      int **ptr_to_ptr = &ptr; // Pointer to a pointer variable
  
      // Accessing the value using pointer to pointer
      printf("Value of num: %d\n", **ptr_to_ptr); // Prints 10
  
      // Modifying the value indirectly using pointer to pointer
      **ptr_to_ptr = 20;
      printf("Modified value of num: %d\n", num); // Prints 20
  
      return 0;
  }

Output:

  Value of num: 10
  Modified value of num: 20

In this example, ptr_to_ptr is a pointer to a pointer that stores the address of ptr.

Dereferencing ptr_to_ptr twice (**ptr_to_ptr) allows us to access the value pointed to by ptr, which is the value of num.

We can modify the value of num indirectly by dereferencing ptr_to_ptr twice and assigning a new value.

Dynamic Memory Allocation:

  • Pointers to pointers are frequently used in dynamic memory allocation scenarios, where memory is allocated dynamically using functions like malloc() and calloc().
  • They allow for the creation of dynamic data structures of arbitrary sizes and shapes.
