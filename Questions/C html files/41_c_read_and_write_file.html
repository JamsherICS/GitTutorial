Reading from a File:

To read from a file in C, follow these steps:

  1. Open the File: Use the fopen() function to open the file in read mode ("r"). This function returns a file pointer that you'll use for subsequent file operations.
FILE *file = fopen("filename.txt", "r");
if (file == NULL) {
    perror("Error opening file");
    return 1;
}
  1. Read Data: Use functions like fscanf() or fgets() to read data from the file into variables or buffers.
char buffer[100];
while (fgets(buffer, sizeof(buffer), file) != NULL) {
    printf("%s", buffer);
}
  1. Close the File: When you're done reading, close the file using fclose() to release system resources.
fclose(file);

Writing to a File:

To write to a file in C, follow these steps:

  1. Open/Create the File: Use the fopen() function to open or create the file in write mode ("w" for overwriting existing content or "a" for appending to existing content).
FILE *file = fopen("filename.txt", "w");
if (file == NULL) {
    perror("Error opening file");
    return 1;
}
  1. Write Data: Use functions like fprintf() or fputs() to write data to the file.
fprintf(file, "Hello, world!\n");
  1. Close the File: When you're done writing, close the file using fclose() to release system resources.
fclose(file);

Example: Reading and Writing to a File:

Here's an example of reading from and writing to a file in C:

#include <stdio.h>

int main() {
    FILE *file;

    // Writing to a file
    file = fopen("example.txt", "w");
    if (file == NULL) {
        perror("Error opening file for writing");
        return 1;
    }
    fprintf(file, "This is an example line written to the file.\n");
    fclose(file);

    // Reading from the same file
    file = fopen("example.txt", "r");
    if (file == NULL) {
        perror("Error opening file for reading");
        return 1;
    }
    char buffer[100];
    while (fgets(buffer, sizeof(buffer), file) != NULL) {
        printf("%s", buffer);
    }
    fclose(file);

    return 0;
}

This example first writes a line to a file named "example.txt" and then reads and prints the contents of the same file. Make sure to handle errors properly and close the file after use to avoid resource leaks.
