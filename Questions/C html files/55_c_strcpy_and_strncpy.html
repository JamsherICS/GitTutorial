strcpy() and strncpy() are both string manipulation functions in C, but they have some key differences, particularly in how they handle copying characters from one string to another, especially when dealing with null-terminated strings.

strcpy() Function:
 Prototype: char *strcpy(char *dest, const char *src);
Copies the entire string from source to destination.
Does not check the length of the source string. It copies characters until it encounters the null terminator ('\0') in the source string.
May cause buffer overflow if the source string is longer than the destination buffer.

Example:

// Using strcpy() to copy source to destination
    strcpy(destination, source);
    printf("After strcpy(), destination contains: %s\n", destination);
    return 0;

strncpy() Function:
 Prototype: char *strncpy(char *dest, const char *src, size_t n);
Copies at most n characters from source to destination.
If the length of the source string is less than n, it pads the destination string with null characters ('\0') until it reaches n characters.
If the length of the source string is greater than or equal to n, strncpy() copies exactly n characters and does not null-terminate the destination string if the source string is not null-terminated within the first n characters.
Useful for ensuring buffer boundaries and preventing buffer overflows.

Example:

// Copy at most 19 characters from source2 to destination2
    strncpy(destination2, source2, 19);
    destination2[19] = '\0'; // Null-terminate the string manually as strncpy() may not do it
    printf("After strncpy(), destination2 contains: %s\n", destination2);
    return 0;

The following C program demonstrates the usage of strcpy() and strncpy() functions:

#include <stdio.h>
    #include <string.h>
    
    int main() {
        char source[] = "Hello, world!";
        char destination[20]; // Destination buffer size should be large enough to hold the copied string
    
        // Using strcpy() to copy source to destination
        strcpy(destination, source);
        printf("After strcpy(), destination contains: %s\n", destination);
    
        // Using strncpy() to copy only a portion of source to destination
        char source2[] = "This is a longer string than the previous one";
        char destination2[20]; // Destination buffer size is limited to 20 characters
    
        strncpy(destination2, source2, 19); // Copy at most 19 characters from source2 to destination2
        destination2[19] = '\0'; // Null-terminate the string manually as strncpy() may not do it
        printf("After strncpy(), destination2 contains: %s\n", destination2);
    
        return 0;
    }
