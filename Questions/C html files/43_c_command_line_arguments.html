Command-line arguments in C are parameters passed to a program when it is executed from the command line. They are typically used to provide input data, specify options, or control the execution of the program.

How to Use Command-line Arguments in C:

Function Signature:
In the main() function, you can declare two parameters: argc (argument count) and argv (argument vector).

int main(int argc, char *argv[])

Argument Count (argc):

  • argc holds the number of command-line arguments passed to the program, including the name of the program itself.
  • It's an integer value that indicates how many arguments were provided.

Argument Vector (argv):

  • argv is an array of strings (char*) that holds the actual command-line arguments.
  • Each element of argv is a null-terminated C string representing one command-line argument.
  • argv[0] contains the name of the program itself, while argv[1] to argv[argc-1] contain the actual arguments provided by the user.

Accessing Command-line Arguments:

You can access command-line arguments using array notation (argv[index]) just like accessing elements of any other array.

printf("Program name: %s\n", argv[0]);

Example:

Below is an example of using command-line arguments in a C program:

#include <stdio.h>
    
    int main(int argc, char *argv[]) {
        printf("Number of arguments: %d\n", argc);
    
        // Print all command-line arguments
        for (int i = 0; i < argc; i++) {
            printf("Argument %d: %s\n", i, argv[i]);
        }
    
        return 0;
    }

How to Run a Program with Command-line Arguments:

To run a C program with command-line arguments, you need to provide them after the program's executable name when invoking the program from the command line. For example:

./program arg1 arg2 arg3

Here, program is the name of the executable file, and arg1, arg2, arg3 are the command-line arguments.

Example Output:

For the above example program, if run with command-line arguments like this:

./program hello world 123

The output would be:

Number of arguments: 4
Argument 0: ./program
Argument 1: hello
Argument 2: world
Argument 3: 123

This demonstrates how command-line arguments can be used to pass information to a C program from the command line.
