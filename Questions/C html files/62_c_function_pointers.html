Declaration of Function Pointer:

Start by declaring a function pointer variable. This is similar to declaring a regular function, but instead of specifying the function's return type and parameter types, you specify the return type and parameter types of the function that the pointer will point to.

Syntax: return_type (*pointer_name)(parameter_types);

Assignment of Function Address:

Assign the address of a function to the function pointer variable. This is done using the address-of operator & followed by the function name.

Example: pointer_name = &function_name;

Indirect Invocation of Function:

Call the function indirectly through the function pointer variable. This is done by dereferencing the function pointer using the dereference operator * or by simply using the function pointer variable name followed by the parameter list in parentheses.

Example: result = (*pointer_name)(arguments); or result = pointer_name(arguments);

Here's a summarized example applying these steps:

  #include <stdio.h>
  
  // Step 1: Declaration of Function Pointer
  int (*operation)(int, int);
  
  // Function declarations
  int add(int a, int b);
  int subtract(int a, int b);
  
  int main() {
      // Step 2: Assignment of Function Address
      operation = &add;
  
      // Step 3: Indirect Invocation of Function
      int result = operation(5, 3);
      printf("Result of addition: %d\n", result);
  
      // Change the function assigned to the function pointer
      operation = &subtract;
      result = operation(5, 3);
      printf("Result of subtraction: %d\n", result);
  
      return 0;
  }
  
  // Function definitions
  int add(int a, int b) {
      return a + b;
  }
  
  int subtract(int a, int b) {
      return a - b;
  }

Output:

  Result of addition: 8
  Result of subtraction: 2
