You can find the largest and smallest elements in an array by iterating through the array and keeping track of the current largest and smallest elements encountered. Here's how you can do it in C:

  #include <stdio.h>
  
  int main() {
      int arr[] = {5, 3, 9, 1, 7, 4, 2};
      int n = sizeof(arr) / sizeof(arr[0]); // Calculate the size of the array
  
      // Initialize variables to store the largest and smallest elements
      int largest = arr[0];
      int smallest = arr[0];
  
      // Iterate through the array
      for (int i = 1; i < n; i++) {
          // Update the largest element if a larger element is found
          if (arr[i] > largest) {
              largest = arr[i];
          }
          // Update the smallest element if a smaller element is found
          if (arr[i] < smallest) {
              smallest = arr[i];
          }
      }
  
      printf("Largest element: %d\n", largest);
      printf("Smallest element: %d\n", smallest);
  
      return 0;
  }

Output:

  Largest element: 9
  Smallest element: 1

In this example:

  • We initialize variables largest and smallest with the first element of the array.
  • We iterate through the array starting from the second element (i = 1) and update largest and smallest as necessary.
  • After iterating through the array, we print the values of largest and smallest.
