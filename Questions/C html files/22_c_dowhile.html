The do-while loop in C is a control flow statement that allows a block of code to be executed repeatedly until a specified condition evaluates to false. Unlike the while loop, the do-while loop evaluates the condition after the execution of the loop body, ensuring that the loop body is executed at least once, even if the condition is initially false.

The general syntax of the do-while loop is as follows:

do {
    // code block to be executed
} while (condition);

In this syntax:

  • The do keyword indicates the start of the loop.
  • The code block within the curly braces {} represents the body of the loop, which contains the statements to be executed.
  • After the execution of the loop body, the while keyword is followed by the condition inside parentheses ().
  • If the condition evaluates to true, the loop iterates and executes the code block again. If the condition evaluates to false, the loop terminates.

Here's an example of using a do-while loop in C to print numbers from 1 to 5:

#include <stdio.h>

int main() {
    int i = 1; // Initialize loop control variable

    // Do-while loop to print numbers from 1 to 5
    do {
        printf("%d ", i); // Print the current value of i
        i++; // Increment i for the next iteration
    } while (i <= 5); // Continue loop while i is less than or equal to 5

    return 0;
}

In this example:

  • We initialize the loop control variable i to 1 before the loop.
  • The do-while loop executes the code block inside the curly braces {} and prints the value of i.
  • After each iteration, i is incremented by 1 using the i++ statement.
  • The loop continues as long as the condition i <= 5 is true.
  • Once i becomes greater than 5, the condition becomes false, and the loop terminates.
