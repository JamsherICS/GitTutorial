A structure in C is a user-defined data type that allows you to group together different types of variables under a single name. Each variable within a structure is called a member or field. Structures are useful for organizing related data in a program.

Here's the general syntax for defining a structure:

struct structure_name {
    data_type1 member1;
    data_type2 member2;
    ...
};

Once a structure is defined, you can declare variables of that type and access its members using the dot (.) operator. Here's an example:

struct Person {
    char name[50];
    int age;
};

// Declare a variable of type Person
struct Person person1;

// Access and modify structure members
strcpy(person1.name, "John");
person1.age = 30;

In this example, we define a structure Person with two members: name and age. We then declare a variable person1 of type Person and access its members to assign values.
