The scanf() function in C is used to read formatted input from the standard input device, typically the keyboard. Its syntax is as follows:


// scanf() function syntax
scanf("format_string", &arg1, &arg2, ...);

format_string: A string that specifies the format of the input to be read. It may contain format specifiers like %d, %f, %s, etc., to indicate the expected type of input.

&arg1, &arg2, ...: Pointers to the variables where the input values will be stored. They must match the format specifiers in the format string and be passed by reference using the address-of operator (&).


// Example usage of scanf()
int num;
printf("Enter a number: ");
scanf("%d", &num);

This will prompt the user to enter a number, which will be stored in the variable num.
