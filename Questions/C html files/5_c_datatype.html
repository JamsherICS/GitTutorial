• int: Used to store integer values (e.g., 5, -10, 1000).
  • float: Used to store floating-point numbers (e.g., 3.14, -0.5, 10.0).
  • char: Used to store single characters (e.g., 'A', 'b', '1').
  • double: Used to store double-precision floating-point numbers (e.g., 3.14159, 2.71828).
  • void: Represents an empty data type (used in function return types and pointers).
  • _Bool: Used to store boolean values (true or false).
  • short, long, long long: Used to store integers of different sizes.
  • unsigned: Used to declare unsigned integer types (non-negative values only).
  • signed: Used to declare signed integer types (positive or negative values).
