In programming, when you pass arguments to a function, you can do so either by value or by reference. The difference lies in how the function accesses and manipulates the values of these arguments:

Pass by Value:
  • When you pass arguments to a function by value, the function receives a copy of the actual value of the arguments.
  • Any changes made to the parameter variables inside the function do not affect the original variables in the calling code.
  • It's like giving someone a photocopy of a document rather than the original document itself. Modifying the photocopy doesn't affect the original document.
Pass by Reference:
  • When you pass arguments to a function by reference, the function receives the memory address (reference) of the actual variables.
  • Any changes made to the parameter variables inside the function directly affect the original variables in the calling code because they both point to the same memory location.
  • It's like giving someone the key to a locked box. Any changes made to the contents of the box will affect the original contents since they share the same space.

Example:

#include <stdio.h>
              
    // Function prototypes
    void passByValue(int x);
    void passByReference(int *x);
      
    int main() {
        int num = 10;
      
        // Pass by value
        printf("Before passByValue: num = %d\n", num);
        passByValue(num);
        printf("After passByValue: num = %d\n", num);
      
        // Pass by reference
        printf("Before passByReference: num = %d\n", num);
        passByReference(&num);
        printf("After passByReference: num = %d\n", num);
        return 0;
    }
              
    // Function to demonstrate pass by value
    void passByValue(int x) {
        x = x * 2;
        printf("Inside passByValue: x = %d\n", x);
    }
              
    // Function to demonstrate pass by reference
    void passByReference(int *x) {
        *x = *x * 2;
        printf("Inside passByReference: *x = %d\n", *x);
    }

Output:

    Before passByValue: num = 10
    Inside passByValue: x = 20
    After passByValue: num = 10
    Before passByReference: num = 10
    Inside passByReference: *x = 20
    After passByReference: num = 20

In this example:

  • In the main() function, we have an integer variable num initialized with the value 10.
  • We then call two functions: passByValue() and passByReference(), passing num as an argument to both.
  • passByValue() takes the value of num (10) as its parameter. Inside the function, it doubles the value of x, but this change doesn't affect the original num variable in the main() function.
  • passByReference() takes a pointer to num as its parameter. Inside the function, it doubles the value pointed to by x, which directly modifies the original num variable.
