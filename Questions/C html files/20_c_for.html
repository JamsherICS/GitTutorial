The for loop is used to execute a block of code repeatedly for a specified number of times. It is often used when the number of iterations is known beforehand.

Syntax:

for (initialization; condition; update) {
    // code block to be executed
}

Initialization: It is an expression used to initialize the loop control variable before the loop starts. This part is executed only once at the beginning of the loop.

Condition: It is a Boolean expression that is evaluated before each iteration. If the condition evaluates to true, the loop continues executing. If it evaluates to false, the loop terminates.

Update: It is an expression used to update the loop control variable after each iteration. It typically increments or decrements the variable to control the loop's behavior.


Here's an example of using a for loop to print numbers from 1 to 5:

#include <stdio.h>

int main() {
    int i; // Loop control variable

    // Loop to print numbers from 1 to 5
    for (i = 1; i <= 5; i++) {
        printf("%d ", i); // Print the current value of i
    }

    return 0;
}

Output: 1 2 3 4 5

In this example:

  • i is initialized to 1 before the loop (int i = 1;).
  • The loop continues as long as i is less than or equal to 5 (i <= 5;).
  • After each iteration, the value of i is incremented by 1 (i++).
