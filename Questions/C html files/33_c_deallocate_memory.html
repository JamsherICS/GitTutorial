Memory allocated dynamically using malloc(), calloc(), or realloc() in C should be deallocated to prevent memory leaks. The free() function is used to deallocate the memory previously allocated by these functions.

#include <stdlib.h>

int main() {
    int *ptr;

    // Allocate memory using malloc()
    ptr = (int *)malloc(5 * sizeof(int));

    // Check if memory allocation is successful
    if (ptr == NULL) {
        printf("Memory allocation failed\n");
        return 1;
    }

    // Use the allocated memory

    // Deallocate memory using free()
    free(ptr);
    ptr = NULL; // Optional: Set pointer to NULL after deallocation

    return 0;
}

In this example, memory is allocated using malloc() and then deallocated using free(). It's good practice to set the pointer to NULL after deallocation to avoid accessing deallocated memory.
