In C, you can open and close a file using the fopen() and fclose() functions, respectively.

Opening a File:

To open a file, you use the fopen() function, which takes two arguments:

  • The file path: A string containing the path to the file you want to open.
  • The mode: A string indicating the mode in which you want to open the file (e.g., read, write, append).

The syntax for fopen() is as follows:

FILE *fptr;
fptr = fopen("file_path", "mode");

For example:

FILE *filePointer;
filePointer = fopen("example.txt", "w");

Closing a File:

After you have finished working with a file, it's important to close it using the fclose() function. This function takes one argument: the file pointer that was returned by fopen().

The syntax for fclose() is simple:

fclose(filePointer);

It's good practice to close files after you're done using them to release system resources and ensure data integrity.

Complete Example:

#include <stdio.h>
  
  int main() {
      FILE *filePointer;
  
      // Open a file named "example.txt" for writing
      filePointer = fopen("example.txt", "w");
  
      // Check if file opened successfully
      if (filePointer == NULL) {
          printf("Error opening file!\n");
          return 1; // Return an error code
      }
  
      printf("File opened successfully!\n");
  
      // Close the file
      fclose(filePointer);
  
      printf("File closed.\n");
  
      return 0; // Return success
  }

In this example:

  • We open a file named "example.txt" for writing using fopen() with mode "w".
  • We check if the file was opened successfully and print a message accordingly.
  • We close the file using fclose().
  • We print a message indicating that the file has been closed.
  • Finally, we return 0 to indicate success.
