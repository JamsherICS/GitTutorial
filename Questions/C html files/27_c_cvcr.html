In C, function arguments can be passed in two ways: call by value and call by reference. Here's the difference between them:

  1. Call by Value:
    • In call by value, a copy of the actual argument is passed to the function parameter.
    • Changes made to the parameter inside the function do not affect the original argument.
    • Simple data types like integers, floats, and characters are passed by value.
  2. Call by Reference:
    • In call by reference, the address of the actual argument is passed to the function parameter.
    • Changes made to the parameter inside the function affect the original argument.
    • Pointers are used to implement call by reference in C.

Here's an example illustrating the difference between call by value and call by reference:

// Call by value function
void changeValueByValue(int x) {
    x = 10; // Changes made to 'x' inside the function don't affect the original argument
}

// Call by reference function
void changeValueByReference(int* x) {
    *x = 10; // Changes made to '*x' inside the function affect the original argument
}

int main() {
    int a = 5;

    // Call by value
    changeValueByValue(a);
    printf("Value after call by value: %d\n", a); // Output: 5

    // Call by reference
    changeValueByReference(&a);
    printf("Value after call by reference: %d\n", a); // Output: 10

    return 0;
}

In this example:

  • changeValueByValue demonstrates call by value, where changes to x inside the function do not affect the original argument.
  • changeValueByReference demonstrates call by reference using pointers, where changes to *x inside the function affect the original argument.
