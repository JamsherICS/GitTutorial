One common way to reverse a string in C is to use a loop to iterate through the string and swap characters from the beginning with characters from the end until reaching the middle of the string.

Here's a simple example of how you can do this:

#include <stdio.h>
#include <string.h>

void reverseString(char* str) {
    int length = strlen(str);
    int start = 0;
    int end = length - 1;

    while (start < end) {
        // Swap characters
        char temp = str[start];
        str[start] = str[end];
        str[end] = temp;

        // Move pointers towards the middle
        start++;
        end--;
    }
}

int main() {
    char str[] = "Hello, World!";
    
    printf("Original string: %s\n", str);
    
    // Reverse the string
    reverseString(str);
    
    printf("Reversed string: %s\n", str);
    
    return 0;
}

Output:

Original string: Hello, World!
Reversed string: !dlroW ,olleH

In this example, the reverseString function takes a pointer to a character array (char* str) as its parameter. It calculates the length of the string using strlen, then iterates through the string using a while loop, swapping characters until the start pointer meets or exceeds the end pointer. Finally, it prints the reversed string in the main function.
