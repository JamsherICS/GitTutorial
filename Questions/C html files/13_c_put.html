putchar() function: Used to print a single character to the standard output.

#include <stdio.h>

int main() {
    putchar('A');  // Prints the character 'A'
    return 0;
}

printf() function: The printf() function is a more versatile function used for formatted output. It can output strings, numbers, and other data types, formatted according to a format string. It can handle multiple arguments and supports various format specifiers for different data types, such as %d for integers, %f for floating-point numbers, and %s for strings.

#include <stdio.h>

int main() {
    printf("The answer is %d\n", 42);  // Prints: The answer is 42
    return 0;
}
