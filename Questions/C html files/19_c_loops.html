1. for loop:

The for loop is used when the number of iterations is known beforehand. It consists of three parts: initialization, condition, and update. The loop iterates as long as the condition is true.

Syntax:

for (initialization; condition; update) {
    // code block to be executed
}

2. while loop:

The while loop is used when the number of iterations is not known beforehand, and the loop continues as long as a specified condition is true. It evaluates the condition before executing the loop body.

Syntax:

while (condition) {
    // code block to be executed
}

3. do-while loop:

The do-while loop is similar to the while loop, but it evaluates the condition after executing the loop body. This guarantees that the loop body is executed at least once, even if the condition is false initially.

Syntax:

do {
    // code block to be executed
} while (condition);

Important points:

  • The for loop is commonly used when the number of iterations is known and requires initialization, condition, and update logic in one place.
  • The while loop is used when the number of iterations is not known beforehand and relies solely on a condition.
  • The do-while loop is useful when you want to execute the loop body at least once, regardless of the initial condition.
