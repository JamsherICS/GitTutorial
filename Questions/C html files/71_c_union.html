In C, a union is a composite data type that allows you to store different types of data in the same memory location. Like a structure, a union enables you to group multiple variables together.

All members of a union share the same memory space, and modifying one member may affect the values of other members.

Here's a comparison between a structure and a union in C:

1. Structure:

  • A structure allows you to group multiple variables of different types under a single name.
  • Each member of a structure has its own separate memory space.
  • Memory is allocated for each member independently, based on its size and type.
  • You can access individual members of a structure independently.

Example:

struct Person {
     char name[50];
     int age;
     float height;
  };

2. Union:

  • A union, like a structure, allows you to group multiple variables of different types under a single name.
  • All members of a union share the same memory space.
  • Memory is allocated for the largest member of the union, and all other members share that memory space.
  • Accessing one member may overwrite the values of other members, as they occupy the same memory location.

Example:

union Data {
  int intValue;
  float floatValue;
  char stringValue[20];
};

Here's how you might use a union:

#include <stdio.h>

union Data {
  int intValue;
  float floatValue;
  char stringValue[20];
};

int main() {
  union Data data;

  data.intValue = 10;
  printf("intValue: %d\n", data.intValue);

  data.floatValue = 3.14;
  printf("floatValue: %.2f\n", data.floatValue);

  strcpy(data.stringValue, "Hello");
  printf("stringValue: %s\n", data.stringValue);

  return 0;
}

Output:

  intValue: 10
  floatValue: 3.14
  stringValue: Hello

In this example, we define a union Data that can hold an integer, a float, or a string. Since all members share the same memory space, modifying one member (for example, intValue) will affect the values of other members (floatValue and stringValue).
