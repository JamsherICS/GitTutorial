Recursion is a programming technique where a function calls itself in order to solve a problem. It's particularly useful for solving problems that can be broken down into smaller, similar subproblems.

#include <stdio.h>

// Example of a recursive function to calculate factorial
int factorial(int n) {
    // Base case: If n is 0 or 1, return 1
    if (n == 0 || n == 1) {
        return 1;
    }
    // Recursive case: Return n * factorial of (n - 1)
    else {
        return n * factorial(n - 1);
    }
}

int main() {
    int n = 5;
    // Calculate factorial of 5 using recursion
    int result = factorial(n);
    printf("Factorial of %d is %d\n", n, result);
    return 0;
}

In this example, the factorial function is defined recursively to calculate the factorial of a non-negative integer n. The function calls itself with a smaller value of n until it reaches the base case where n is 0 or 1.
