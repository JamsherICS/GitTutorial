In C, you can find the length of a string using the strlen() function, which is available in the <string.h> header file.

Here's how you can find:

#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello, World!";
    
    // Using strlen() function
    int length = strlen(str);
    
    printf("Length of the string: %d\n", length);
    
    return 0;
}

Output:

Length of the string: 13

In this example, the strlen() function takes a pointer to a null-terminated character array (string) as its argument and returns the length of the string excluding the null character \0. The length is then printed to the console.
