A pointer in C is a variable that stores the memory address of another variable. Pointers are powerful features of C that allow for dynamic memory allocation, passing of addresses instead of values, and working with complex data structures.

Here's how you declare a pointer variable:

data_type *pointer_name;

Here's an example of declaring and using pointers:

int number = 10;
int *ptr;  // Declare a pointer to an integer

ptr = &number;  // Assign the address of 'number' to the pointer

printf("Address of number variable: %p\n", ptr);  // Print the address stored in the pointer
printf("Value of number variable: %d\n", *ptr);  // Print the value pointed to by the pointer

In this example:

  • We declare an integer variable number and initialize it to 10.
  • We declare a pointer variable ptr to an integer.
  • We assign the address of the number variable to the pointer using the & (address-of) operator.
  • We print the address stored in the pointer using the %p format specifier.
  • We print the value pointed to by the pointer using the %d format specifier and the * (dereference) operator.
