1. Print Statement vs. Print Function:

Python 2: Uses the print statement. Example: print "Hello, world!"

Python 3: Uses the print function. Example: print("Hello, world!").

2. Integer Division:

Python 2: Integer division truncates the decimal part and returns an integer. Example: 5 / 2 results in 2.

Python 3: Integer division returns a float. Example: 5 / 2 results in 2.5. To perform integer division, you use //. Example: 5 // 2 results in 2.

3. Exception Handling:

Python 2: Exceptions are handled using except Exception, e:.

Python 3: Exceptions are handled using except Exception as e:.
